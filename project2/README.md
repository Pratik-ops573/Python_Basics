# Python Student Record Projects

This repository contains two simple Python projects for managing student records.  
These projects are designed for beginners to practice Python programming, file handling, and database operations.

---

## Project 2: Simple Student Record System

A console-based program to manage student marks.

### Features
- **Add/Update Marks:** Enter or modify marks for a student.
- **View Result:** Display marks and results of a student.
- **Exit:** Close the program.

### How to Run
1. Ensure Python is installed on your system.
2. Clone or download this repository.
3. Open a terminal in the project directory.
4. Run the program:

```bash
   python student_record.py
