# Student Record System Using Database

A simple Python program to manage student information and marks using a MySQL database.  
This project demonstrates basic **CRUD operations** (Create, Read, Update, Delete) and report generation for students.

---

## Features
- **Add / Update Student:** Add a new student or update an existing student record.
- **View Student:** Display details and marks of a specific student.
- **Print Report:** Generate a text file report for a student.
- **Delete Student:** Remove a student from the database.
- **Exit:** Close the program.

---

- **Python 3.x**
- **MySQL Database**
